# DevFolio Dark — Template Portfólio Programador

Template de portfólio premium para desenvolvedores e freelancers de tech.

## Arquivos
- `site/index.html` — página principal
- `site/style.css` — tema dark neon green/purple
- `site/script.js` — typing effect, cursor, skill bars, animações
- `site/assets/icons/favicon.svg`

## Personalização rápida
1. Substitua `Alex Fernandes` pelo seu nome
2. Troque `LINK_CAKTO_AQUI` pelo link de contratação/Calendly
3. Edite skills, projetos e experiência profissional
4. Atualize e-mail e WhatsApp
