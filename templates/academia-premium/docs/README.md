# Academia Premium — Template IronForce Gym

Template de landing page premium para academias e centros de treinamento.

## Arquivos inclusos

```
site/
├── index.html         — página principal
├── style.css          — todos os estilos
├── script.js          — interações e animações
└── assets/
    └── icons/
        └── favicon.svg
```

## Como usar

1. Substitua `LINK_CAKTO_AQUI` pelo link de checkout real
2. Troque `5511999999999` pelo número do WhatsApp
3. Edite os textos, preços e informações de contato
4. Suba os arquivos para qualquer hospedagem (Netlify, Vercel, GitHub Pages, etc.)

## Tecnologias

- HTML5 semântico
- CSS3 puro (variáveis, grid, flexbox, animações)
- JavaScript puro (sem dependências)

## Compatibilidade

Chrome, Firefox, Safari, Edge — desktop e mobile.
